手で解く用のプログラムです。

このフォルダ内の、1つのソースファイルが独立した作品（プログラム）となっており、それぞれ以下のようになっています。

・15puzzle.cpp            … 15puzzle. スライドパズルといえばこれが有名.
・3kakuPuzzle.cpp　　　　 … 正三角形版パズル
・6kakuPuzzle_noAnime.cpp … 正六角形版パズル（滑らかなアニメーションなし）


工夫点は特にありませんが、強いて言えば、アニメーション「滑らかにブロックを動かす」です。

例えば、距離1離れた場所に移動させるときに
0.9, 0.8, 0.7…のように近づけるのではなくて
0.5, 0.25, 0.125…のように近づけると、なんとなくヌルっとして気持ちよかったので、そのようなアニメーションにしてみました。
（移動フレーム数が12を超えると強制的に定位置にセットされるため、あんまりヌルッとしないかも？）

実装は、多少汚いです。（ごめんなさい）

ーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

動作環境
・Dxライブラリ
・Visual Studio 2017

使用言語
C++

環境構築／実行方法など
・Visutal Studioで、Dxライブラリのサンプルコードが動く環境を構築する
・例えば、15puzzle.cppをプロジェクトに追加し、ビルドする
・実行される

注意点
例えば、
15puzzle.cpp
3kakuPuzzle.cpp
を一緒にビルドするとエラーが出ます。それぞれがmain関数を持った、独立したアプリケーションであるためです。

ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

遊び方：
マウスで空白の隣のマスをカチカチするとブロックが移動します。
